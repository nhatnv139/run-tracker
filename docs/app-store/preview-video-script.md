# RunVie — App Preview Video Script & Storyboard

> Duration: 30 giây (Apple App Preview limit) | Format: vertical 1080×1920 + landscape 1920×1080 versions
> Delivery: Apple App Preview + Google Play Promo Video + paid social cut-downs (15s, 6s)
> Owner: Motion Design Lead + Music Producer | Deadline: W-3 trước launch

---

## 1. Strategic frame

**Single message**: "RunVie hiểu người Việt — từ bữa phở đến từng bước chạy."

**Tone**: Optimistic, warm, anchored in everyday Vietnamese life. Avoid stock-fitness clichés (gym, abs, six-pack).

**Emotional arc**:
```
Curiosity (logo) → Confidence (run) → Connection (AI coach) → Achievement (badge) → 
  Reward (voucher) → Belonging (community) → Action (CTA)
```

---

## 2. Music & sound brief

| Element | Spec |
|---------|------|
| Genre | Vietnamese instrument fusion — electronic |
| BPM | 128 (steady, running-cadence aligned) |
| Key | C minor (warm but driven) |
| Instruments | Đàn tranh (Vietnamese zither) lead melody 0–15s, sub-bass synth 808 throughout, đàn bầu accent at 22s, electronic kick layered |
| Reference tracks | Onra "Long Distance"; Hoaprox "Ngẫu Hứng"; ODESZA "A Moment Apart" energy |
| Mix | Music 70%, voice 95% (Vietnamese voiceover priority), SFX 40% |
| Royalty | MUST original composition or fully cleared sync license, no library tracks for Apple Editorial pitch |

Sound design beats:
- 0s: subtle door-knock thud as logo lands
- 3s: whoosh transition into run screen
- 8s: AI Coach gentle chime when chat appears
- 15s: voucher unlock "ka-ching" subtle, never tacky
- 22s: cheering crowd reverb tail (community)
- 28s: clean cut to silence + Aurora gradient pulse

## 4. Storyboard 30 giây

### Frame 1 — 0.0s to 3.0s — Logo Splash

| Spec | Detail |
|------|--------|
| Visual | Background Aurora gradient pulse (Coral → Mint → Lavender diagonal). Logo RunVie wordmark center, scale 0.8→1.0 ease-out 800ms. Underneath wordmark, tagline fade-in 1500ms `Chạy bộ thông minh.` |
| Camera | Static, no movement |
| Audio | Subtle thud at 0.4s when logo lands. Đàn tranh single note rising. |
| Text overlay | None |
| Transition out | At 2.8s, gradient swirl → wipe diagonal to next scene |

### Frame 2 — 3.0s to 8.0s — Run Screen Hero

| Spec | Detail |
|------|--------|
| Visual | iPhone 15 Pro Max mockup tilted 8° right. Screen shows run active: `5.42 km` big number animates from `0.00` counting up to `5.42` in 1.5s. Map route Hồ Tây behind, polyline drawing in. |
| Camera | Slow push-in 5% scale across 5s |
| Audio | Đàn tranh melody full intro. Run footsteps SFX subtle. VO at 5s: "Năm cây số xong, pace 5'42 — cố lên Mai!" |
| Text overlay | Top: `AI Coach tiếng Việt 24/7` font Be Vietnam Pro Bold 72pt, fade-in at 5.5s |
| B-roll insert | Optional: 0.5s split-screen with real runner POV — feet on Hồ Tây path, sunrise — only if budget allows |

### Frame 3 — 8.0s to 15.0s — Quick Cuts Montage

7 seconds, 4 micro-scenes at ~1.75s each:

| Sub | Time | Visual |
|-----|------|--------|
| 3a | 8.0–9.7s | AI Coach chat screen — Vietnamese text bubbles animate up. Highlight bubble "Đầu gối hơi đau? Mai mình đi bộ nhẹ thôi nhé." (matches VO at 12s) |
| 3b | 9.7–11.4s | Heatmap year view scroll-reveal — 365 ô lighting up sequentially Mint → Coral, ends on `Streak: 47 ngày` |
| 3c | 11.4–13.1s | Badge unlock animation — circular medal Bát Tràng 3D rotating, text "Thử thách Tết · 100km" appears |
| 3d | 13.1–15.0s | Step counter ring filling 0% → 82%, big number `8.247 bước`, hand entering frame holding phone |

Audio: BPM rises, kick drum enters at 11s. VO at 12s.
Text overlay: minimal — each cut has small caption bottom-left.

### Frame 4 — 15.0s to 22.0s — Voucher Unlock

### Frame 5 — 22.0s to 28.0s — Community

| Spec | Detail |

### Frame 6 — 28.0s to 30.0s — Logo + CTA

| Spec | Detail |
|------|--------|
| Visual | Cut to clean Aurora gradient background. Logo RunVie center, tagline `Chạy bộ thông minh.` Below, CTA button `Tải miễn phí` Coral fill, white text. App Store + Play Store badges below CTA, side by side. |
| Camera | Static |
| Audio | Music outro — final đàn tranh note + reverb tail. VO at 29s. |
| Text overlay | Logo + CTA only |

## 5. Cut-downs

### 15-second version (Apple Search Ads + Meta Reels)
Frames 1 (2s) → 2 (4s) → 3 (4s, only sub 3a + 3c) → 4 (3s) → 6 (2s). Drops Frame 5.

### 6-second version (TikTok pre-roll)
Frames 2 (2s) → 4 (2s) → 6 (2s). Pure reward-loop story.

### Vertical 9:16 vs Landscape 16:9
- Vertical: default. Apple App Preview accepts portrait.
- Landscape: only for YouTube ads + landing page hero.

## 8. Compliance checks

- [ ] All real runners signed model release with talent fee documented
- [ ] No competitor logos visible (no Garmin, Polar, Apple Fitness, Strava watermark)
- [ ] Voucher amounts shown are placeholder, NOT a promise — small disclaimer `Hình ảnh minh hoạ` 12pt bottom-right at 18-20s
- [ ] No health claims ("giảm 5kg trong 30 ngày") — Apple Guideline 1.4.1 + Vietnamese consumer law
- [ ] All Vietnamese spellings reviewed by linguistics QA
- [ ] Music sync license signed before final export
- [ ] Caption file SRT exported for accessibility (Vietnamese + English)
